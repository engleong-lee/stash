# Screenshot Guide for Chrome Web Store

Chrome Web Store requires at least 1 screenshot (up to 5), size: 1280x800 or 640x400

- Make sure the extension looks polished

## Tips for Great Screenshots

1. **Use a clean browser** - Close unnecessary tabs
2. **Use meaningful session names** - "React Documentation Review", "Project Research", etc.
3. **Test in both light and dark mode** - Consider including a dark mode screenshot
4. **Make sure Ollama is running** - Show the "âœ“ Running" status
5. **Use a high-DPI display** if possible for crisp screenshots

## How to Capture

### macOS
- `Cmd + Shift + 4` then drag to select area
- Or `Cmd + Shift + 5` for more options

### Windows
- `Win + Shift + S` to open Snip & Sketch
- Select the area to capture
